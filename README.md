***YO BRO 
1.**Clone the repo**

```
$ git clone git@github.com:Precho/Crypto-RN.git
$ cd Crypto-RN
```
2.**Install dependencies**

```
$ npm install
$ brew install watchman
```
3.**Start Packager**

```
$ npm start
```
4.**Running on iOS:**

```
$ react-native run-ios
```
5.**Running on Android:**

```
$ react-native run-android
```
